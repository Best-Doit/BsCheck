# Sprints BsCheck (MVP)

Este directorio almacena el historial vivo de los **sprints** del proyecto BsCheck para el MVP Android.

Convenciones:

- Un archivo por sprint: `sprint-01.md`, `sprint-02.md`, etc.
- Cada sprint incluye:
  - Objetivo del sprint
  - Alcance (historias / tareas)
  - Estado (Planned / In progress / Done)
  - Notas y decisiones
- El lenguaje de negocio se mantiene en **español**, los nombres técnicos de carpetas y código en **inglés**.
